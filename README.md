# poll_system_api
It is a poll system API based on graphql and Django

## Installation:
$cd Polls
$pip3 install pipenv
$pipenv shell
$pip3 install django
$pip3 install graphene-django
$python manage.py makemigrations
$python manage.py migrate
$python manage.py runserver
